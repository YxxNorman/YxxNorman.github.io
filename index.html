5.3 滤波器

2024-06-20
学习 / 测试技术

滤波器

理想滤波器

  • 截止频率：$f_{c}$
  • 建立时间：$T_{e} = t_{b} - t_{a} = \frac{0.61}{f_{c}}$，$T_{e} = t_{b}’ - t_{a}’ = \frac{0.45}{f_{c}}$
  • 带宽$B$：$T_{e}B = const$

实际滤波器

  • 截止频率：$f_{c1}, f_{c2}$，对应的幅值为$\frac{A_{0}}{\sqrt{2}}$，衰减-3dB
  • 中心频率：$f_{0}$
  • 带宽$B$：$B = f_{c2} - f_{c1}$，决定频率分辨力，带宽越大，频率分辨力越低
  • 品质因素$Q$：$Q = \frac{f_{0}}{B}$
  • 滤波器因素$\lambda$：$\lambda = \frac{B_{-60dB}}{B_{-3dB}}$

直流电桥

电桥灵敏度

连接方式

  • 半桥单臂：$S_{B} = \frac{U_{o}}{\Delta R_{0}/R_{0}}= \frac{\frac{\Delta R_{0}}{4R_{0}}U_{i}}{\Delta R_{0}/R_{0}}=\frac{1}{4}U_{i}$
  • 半桥双臂：$S_{B} = \frac{U_{o}}{\Delta R_{0}/R_{0}}= \frac{\frac{\Delta R_{0}}{2R_{0}}U_{i}}{\Delta R_{0}/R_{0}}=\frac{1}{2}U_{i}$
